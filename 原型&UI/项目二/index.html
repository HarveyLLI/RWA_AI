如何参与金库

连接钱包后，在首页「期限金库大厅」选择到期日，点击「立即存入」。按提示授权 USDT，即可获得对应 wlUSDT-[Date] 生息凭证。存续期内 NAV 按日增长，到期可 1:1 兑付 USDT 本息。

三种退出方式

持有至到期：在金库到期后，于「我的持仓」点击「到期赎回」拿回 USDT。DEX 卖出：将 wlUSDT 转入钱包后在 Uniswap 等 DEX 卖出。兑换 $ANTA：在「我的持仓」点击「兑换 $ANTA」，确认免责后按金库折扣价换取 $ANTA（存在代币波动风险，详见弹窗声明）。

API 与 AI Agent
