连接 AI

AI Agent 可通过 Skills 发现产品与监控收益，通过 Trading API 在用户授权下执行交易。支持 REST API（API Key 鉴权）与 MCP（Model Context Protocol）。

Skills — 数据发现与监控

AI Agent 可调用以下 Skills 获取金库收益数据、NAV、$ANTA 折扣与 DEX 价格，实现程序化发现产品与收益监控。

📊

get_vault_list

返回所有在售金库列表，包含到期日、当前 TVL、凭证代币名称。用于产品发现。

GET /v1/vaults
📈

get_yield_curve

返回各金库的到期年化 YTM、当前 NAV、$ANTA 折扣率、DEX 现货价，供 Agent 比对最优策略。

GET /v1/vaults/yield-curve
💰

get_vault_apy

查询指定金库的实时 APY、30 日回报、成立以来表现等收益指标。

GET /v1/vaults/:id/apy
🔗

get_spot_price

获取凭证在 DEX 上的实时市价，以及与 NAV 的溢价/折价率，用于退出时机判断。

GET /v1/vaults/:id/spot
🛡️

get_risk_metrics

查询底层抵押 BTC 的 LTV、清算阈值、抵押覆盖率、托管方信息，用于风控监控。

GET /v1/risk/ltv
📋

get_custody_proof

获取链上托管证明哈希与验证链接，供 Agent 自动校验资产安全状态。

GET /v1/custody/proof

Trading API — 交易接入

用户连接钱包后，AI Agent 可在用户授权下通过以下接口执行存入、赎回与 $ANTA 兑换等交易操作。

方法 接口 说明 鉴权
POST /v1/vaults/:id/deposit 向指定金库存入 USDT，铸造等值 wlUSDT 凭证。需传入金额与用户签名授权。 钱包签名
POST /v1/vaults/:id/redeem 到期后赎回 wlUSDT 凭证，按 NAV 1:1 兑付 USDT 本息。需用户确认交易。 钱包签名
POST /v1/vaults/:id/swap-anta 将 wlUSDT 凭证按金库折扣价兑换为 $ANTA 股票代币。需用户阅读风险提示并签名确认。 钱包签名
GET /v1/positions 查询当前用户所有持仓，包括各金库凭证余额、当前价值、预估到期收益。 API Key
GET /v1/vaults/:id/nav-history 获取指定金库的历史 NAV 数据，用于绘制净值走势图或计算回报。 API Key

认证与授权

所有只读接口使用 API Key 鉴权（请求头 X-API-Key）。交易类接口需用户连接钱包后，由 AI Agent 构造交易并引导用户在钱包中签名确认。
