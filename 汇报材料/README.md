# 汇报材料

本目录存放向老板汇报用的**完整结构说明**与**飞书汇报文档内容稿**。

| 文件 | 说明 |
|------|------|
| **2026-02-26-老板汇报-完整结构.md** | 汇报不只「展示网页」而应包含的完整模块（业务简介、角色图、网站关系、亮点清单、AI 支持、分阶段目标等），以及各模块放在「飞书文档」还是「现场 demo」的说明。 |
| **2026-02-26-飞书汇报文档-内容稿.md** | 可直接复制到**飞书文档**的全文内容；文中 **[此处插入…]** 为需要您补充的**图**或**链接**位置。飞书支持图、文字、链接，便于老板查阅与分享。 |

**使用方式**：  
1. 在飞书新建一篇文档。  
2. 将《飞书汇报文档-内容稿》全文复制进飞书。  
3. 在标注 **[此处插入业务架构图]**、**[此处插入链接：Antalpha 官网]** 等处，插入实际图片或链接。  
4. 汇报时按《老板汇报-完整结构》中的顺序讲解，网站 demo 部分现场打开并对照飞书中的「网站关系」与「亮点功能清单」讲解。

*若汇报时间或老板反馈有调整，可在此目录增补新版本内容稿。*
