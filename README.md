# RWA&AI 网站 Demo 项目

本项目用于 RWA（Real World Assets）与 AI 相关产品的网站 demo 开发与协作管理。

## 文件夹结构说明

| 目录 | 说明 |
|------|------|
| **业务资料** | 您提供的业务背景、项目信息、补充说明等原始记录，便于后续整理成需求或竞品文档 |
| **竞品调研** | 竞品分析、市场调研文档 |
| **需求文档** | 产品需求、功能说明、需求框架 |
| **任务管理** | 开发任务总表、迭代计划 |
| **原型&UI** | 线框图、高保真原型、UI 资源（可按 官网/应用端/管理后台 分子目录） |
| **执行代码** | 网站 demo 的源代码与工程 |
| **测试用例** | 功能测试、端到端测试用例（可按模块分子目录） |
| **实施部署** | 部署方案、环境与运维文档（可选） |

结构参考自 Law-system 项目，便于统一管理与后续扩展。

## 使用说明

- 需求与背景会陆续补充到 **需求文档** 与 **竞品调研**。
- 开发产出放在 **执行代码**，原型与设计稿放在 **原型&UI**。
- 需要补充信息时可直接在对应文档或对话中说明。
