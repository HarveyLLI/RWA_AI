# 项目二：基金会发行产品站 — 竞品调研

> 整理日期：2026-02-26  
> 用途：为「以基金会发行产品为主导」的独立网站提供竞品参考与可借鉴优势；项目二侧重投资者端的产品列表、募资与 Earn 体验。

---

## 一、项目二定位回顾

- **主体**：基金会（发行方）主导的**独立网站**。
- **核心职责**：发行 RWA/债类产品、链上募资、向投资者展示并销售 Earn 产品（如质押借贷、矿机贷、电费贷等打包为债的产品）。
- **与项目一关系**：项目一（Antalpha 官网 Web3 Infrastructure）讲基础设施与风险能力；项目二是基金会自己的「产品官网」，列出产品、募资入口、透明度与合规信息。

因此，竞品选择标准为：**以产品/池子列表为核心、面向投资者展示收益与风险、具备发行/募资或嵌入式 Earn 能力的平台**。

## 三、Morpho 重点调研（https://morpho.org/）

### 3.1 定位与一句话

- **Slogan**：*Connect to the universal lending network* / *Access global liquidity at the best possible terms powered by open infrastructure that serves, not extracts.*  
- **对项目二启示**：强调「网络」「开放基础设施」「服务而非榨取」，基金会站可强调「由 Antalpha Prime 基础设施支撑、透明、服务投资者」。

### 3.2 官网信息架构（可借鉴）

1. **首屏**  
   - 大标题 + 副标题，强调「连接」「流动性」「开放」。  
   - 核心数据：Deposits / Loans（当前为 $0，可替换为 TVL、在管规模、产品数等）。  
   - 明确 CTA：Scroll to explore / Launch App。

2. **Powered by Morpho**  
   - 独立区块：「Enterprises connect with Morpho to power any lending or borrowing use case at scale.」  
   - **与项目二对应**：可做「Powered by Antalpha Prime」或「由 Antalpha Prime 提供技术与风控」，突出基金会是发行与募资方，底层是同一套基础设施。

3. **三条产品化价值（Built for scale）**  
   - **Embed custom earn products**：任意资产、任意风险与合规要求的收益产品嵌入。→ 链接 [Earn 产品页](https://morpho.org/earn-products/)。  
   - **Facilitate crypto backed loans**：接入全球流动性、即时贷款、保留产品体验所有权。→ 链接 [Coinbase Loans 案例](https://morpho.org/coinbase-loans/)。  
   - **Become a leading curator**：通过 Curate 打造策略、规模化借贷。→ 链接 [Curate 文档](https://docs.morpho.org/curate/)。  
   **启示**：基金会站可把「产品类型」拆成 3–4 条清晰价值（如：质押借贷 Earn、矿机贷、电费贷、多资产池），每条配短说明 + 跳转产品列表或详情。

4. **Open by default, secure by design**  
   - 用**合约代码片段**（Morpho V1 核心）直接放在页面上，强调可验证、透明。  
   - 审计与形式化验证：Cantina、ChainSecurity、OpenZeppelin、Spearbit、Trail of Bits、Certora、Web3Soc 等。  
   **启示**：基金会站可设「透明与安全」区块：链上合约/池子可查、审计报告、Antalpha 风控与预言机说明，增强信任。

5. **Backed by / 生态**  
   - 「Backed by leaders reshaping finance」+ 40+ 机构 logo。  
   - **启示**：若有合作机构、托管方、审计方，可做「合作与背书」区块，提升机构感。

6. **底部 CTA**  
   - 「Unlock the potential of onchain lending」+ Contact us。  
   - **启示**：项目二可做「开始投资」/「查看产品」+「联系商务」双 CTA。

### 3.3 Morpho 产品线与生态（与基金会对照）

| Morpho 侧 | 对基金会站的借鉴点 |
|-----------|---------------------|
| **Consumer：Earn / Borrow** | 投资者端明确区分「赚取收益」与「借款」，产品列表按类型/风险分层。 |
| **Prime：Curate / Rewards** | 类似「精选池/策略」与激励；基金会可做「精选产品」「推荐池」或「新上线」。 |
| **Infra：API / SDK** | 对外暴露接口给第三方嵌入；基金会站可提供「API/数据」入口给 AI 或合作方，与项目一开发者门户呼应。 |
| **Embed custom earn products** | 白标/嵌入收益产品；基金会本身就是「发行方站点」，可强调「合规、透明、一键参与」。 |
| **Curator 无代码创建 Vault** | 发行方快速上架产品；基金会对应「产品上架流程」与「池子参数」的透明展示。 |

### 3.4 可借鉴优势小结（Morpho）

- **信息分层清晰**：首屏 → Powered by → 三条价值 → 透明/安全 → 生态 → CTA。  
- **产品表述 B2B2C**：既给企业（嵌入/合作），又给终端用户（Earn/Borrow）；基金会站可同时面向机构投资者与合格个人。  
- **信任建设**：代码可见、多机构审计、形式化验证、机构背书。  
- **文档与子页**：Earn、Coinbase Loans、Curate 各有独立页或文档，产品线不堆在一页。  
- **设计风格**：简洁、数据突出、深色/专业感，与 Antalpha 官网可保持同一调性。

## 五、可借鉴优势汇总（供项目二页面与产品设计）

| 维度 | 借鉴来源 | 建议落地 |
|------|----------|----------|
| **首屏** | Morpho | 一句价值主张 + 核心数据（TVL/产品数/累计收益）+ 主 CTA（查看产品 / 开始投资）。 |
| **产品分类与价值条** | Morpho（Earn / Loans / Curate） | 3–4 条产品线（如质押借贷、矿机贷、电费贷、多资产池），每条一句话 + 跳转列表或代表产品。 |
| **Powered by / 基础设施** | Morpho | 明确「由 Antalpha Prime 提供技术与风控」，链回项目一或透明度看板。 |
| **产品列表与卡片** | Centrifuge、Maple、Ondo | 每只产品：名称、类型、APY/收益区间、期限、抵押/托管要点、最低起投、CTA「查看详情」/「参与」。 |
| **透明与安全** | Morpho、Centrifuge | 区块：链上可查、审计报告、托管与风控（Antalpha 预言机/尽调），可选展示合约或池子地址。 |
| **投资者准入** | Ondo、Centrifuge | KYC/合格投资者说明、准入流程、赎回与到账时间，放在产品详情或统一「参与须知」。 |
| **机构与背书** | Morpho（Backed by） | 合作机构、托管方、审计方 logo 与简短说明，增强信任。 |
| **文档与 API** | Morpho（API/SDK）、项目一 | 「开发者/数据」入口：池子数据、APY 接口，供 AI 或合作方对接，与项目一文档站区分（项目二偏产品与募资）。 |
| **多语言与合规话术** | 全竞品 + 项目一 | 中英切换、RWA/收益相关合规用语（与项目一免责声明一致）。 |
