# 项目三 — WorldLend 管理平台原型

管理后台原型，用于配合**项目二**（用户端产品站）的运营，管理产品信息、内容与监控数据。

## 如何打开

- 直接用浏览器打开 `index.html`，或通过本地静态服务器（如 `npx serve .`）访问。
- 登录：用户名 `admin`、密码任意（原型演示可留空或输入 `demo`），点击「登录」进入主界面。

## 功能模块（原型范围）

| 模块 | 说明 |
|------|------|
| **仪表盘** | 协议 TVL、在售金库数、今日存入笔数、风控状态；监控告警列表（LTV、NAV、资金利用率等）。 |
| **产品（金库）管理** | 金库列表（名称、代币、到期日、APY、TVL、状态、操作）；「新增金库」与「编辑」进入编辑页。 |
| **编辑金库** | 基础信息（名称中英、代币符号、到期日、网络、上架状态）；收益与折扣（APY、$ANTA 折扣）；合约与链上（金库合约地址、凭证代币地址）；风控与披露（资金需求方、抵押覆盖率、LTV、清算阈值、托管方、链上证明哈希）。 |
| **内容与运营** | 首页/全局统计文案；公告与风险提示（存入说明、兑换 $ANTA 风险提示）的编辑。 |
| **交易监控** | 交易流水表：类型（存入/赎回/兑换 $ANTA）、金库、用户地址、金额、时间、TxHash；筛选（类型、金库、时间范围）。 |
| **系统设置** | 当前登录用户与角色；操作日志（最近记录）。 |

## 与项目二的关系

- 管理端配置的**金库信息、收益/折扣、风控披露、合约地址**等，通过后端 API 提供给项目二前端展示。
- **内容与运营**中的公告与风险提示，可对应项目二弹窗与帮助页文案。
- **交易监控**数据来源于链上事件索引，项目二「最近交易」可调用同一后端 API。

## 技术说明

- 纯静态 HTML + CSS + JS，无构建；路由为 Hash（`#dashboard`、`#vaults`、`#vault-edit` 等）。
- 侧栏导航、登录/登出、页面切换与「编辑/返回列表」已做基础联动，数据为占位，实际开发时接入项目三后端 API。
